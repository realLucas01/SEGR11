# Kurzfassung – Kommerzielles Angebotsdokument

## Produktüberblick
Das **Inflight Multiplayer „4-Connect System“** ermöglicht Passagieren, während des Fluges das Spiel „4 Gewinnt“ auf ihren Sitzmonitoren gegeneinander zu spielen. Die Software integriert sich nahtlos in bestehende Inflight-Entertainment-Systeme (Panasonic, Thales, Wireless IFE).
