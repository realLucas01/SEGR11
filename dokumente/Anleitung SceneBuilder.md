## SceneBuilder:

Eclipse Anleitung:

Eclipse -> Help -> Eclipse Marketplace -> e(fx)clipse installieren

SceneBuilder Installieren von Gluonhq(auf Version achten, muss mit der Java Version übereinstimmen!!!) https://github.com/gluonhq/scenebuilder/releases?page=2

SceneBuilder einbinden: Window ->Preferences -> 'search' -> JavaFx -> SceneBuilder executable -> Pfad eingeben

Verwendeung von SceneBuilder: Im Projekt rechtsklick auf eine .fmxl Datei -> open with SceneBuilder
  
InteliJ:

SceneBuilder Installieren von Gluonhq,( auf Version achten!!!) in unserem Fall 21 https://github.com/gluonhq/scenebuilder/releases?page=2
